Este encontro trabalha **Ambiente de desenvolvimento (editor, navegador, DevTools)** com foco em fundamentos de Web/HTML e autonomia de estudo.

## Conceitos essenciais
- Configuracao do editor de codigo e pastas.
- Uso de navegador e DevTools para depuracao.
- Rotina de salvar, testar e corrigir iterativamente.

## Explicacao detalhada dos conceitos essenciais
### 1) Configuracao do editor de codigo e pastas.
Padrao de pastas, nomes e ferramentas de edicao evita retrabalho e melhora consistencia.

#### Exemplo aplicado
```text

```

### 2) Uso de navegador e DevTools para depuracao.
DevTools permite inspecao do HTML/CSS, leitura de erros e analise de rede.

#### Exemplo aplicado
```js

```

### 3) Rotina de salvar, testar e corrigir iterativamente.
Ciclos curtos de salvar-testar-corrigir evitam acumulo de erro e aceleram aprendizado.

#### Exemplo aplicado
```text
Aplique o conceito no seu projeto e valide no navegador.
```

## Como estudar este encontro sozinho
1. Leia a explicacao geral e os conceitos com atencao.
2. Rode os exemplos e altere valores/propriedades para testar entendimento.
3. Reproduza sem copiar e compare com seu resultado.
4. Resolva uma variacao do tema no seu contexto.

## Erros comuns de iniciantes
- Avancar sem consolidar a base do encontro.
- Copiar codigo sem compreender cada linha.
- Nao validar comportamento no navegador a cada alteracao.
