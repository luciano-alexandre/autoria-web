# Encontro 7 - Imagens, tabelas e organização de conteúdo

**Unidade:** Unidade 1  
**Carga prevista:** 1,5h  
**Entregavel previsto:** Página com mídia e tabela

## Explicacao geral do encontro
Este encontro trabalha **Imagens, tabelas e organização de conteúdo** com foco em fundamentos de Web/HTML e autonomia de estudo.

![Imagem de apoio 1](https://upload.wikimedia.org/wikipedia/commons/6/61/HTML5_logo_and_wordmark.svg)

## Conceitos essenciais
- Insercao de imagens com alt e dimensoes adequadas.
- Estruturacao correta de tabelas para dados.
- Organizacao visual para leitura de conteudo misto.

## Explicacao detalhada dos conceitos essenciais
### 1) Insercao de imagens com alt e dimensoes adequadas.
Texto alternativo descreve imagem para leitores de tela e para cenarios sem carregamento visual.

#### Exemplo aplicado
```html
<img src="turma.jpg" alt="Alunos em laboratorio" width="320" />
<table><tr><th>Item</th><th>Valor</th></tr><tr><td>Aula</td><td>1</td></tr></table>
```

### 2) Estruturacao correta de tabelas para dados.
Tabela deve ser usada para dados tabulares com cabecalho claro e estrutura consistente.

#### Exemplo aplicado
```html
<img src="turma.jpg" alt="Alunos em laboratorio" width="320" />
<table><tr><th>Item</th><th>Valor</th></tr><tr><td>Aula</td><td>1</td></tr></table>
```

### 3) Organizacao visual para leitura de conteudo misto.
Combinar texto, imagem e tabela exige hierarquia visual para manter clareza.

#### Exemplo aplicado
```text
Aplique o conceito no seu projeto e valide no navegador.
```

## Exemplo principal da aula
