# Plano de Aula - Unidade 1 (HTML5 Semântico)

Carga horária: 18h (6 encontros)

## Objetivos da Unidade
- Compreender o papel do HTML na construção de páginas web.
- Criar documentos HTML estruturados com semântica.
- Introduzir acessibilidade e SEO básico na marcação.

## Conteúdos
- Estrutura de documento HTML5.
- Elementos textuais e de agrupamento.
- Hiperlinks, listas, tabelas e mídia.
- Formulários HTML.
- Semântica, acessibilidade inicial e SEO on-page.

## Roteiro por Encontro

### Encontro 1
- Tema: Introdução à Web e ferramentas
- Prática: criar estrutura de pastas do projeto e primeiro `index.html`
- Produto: página com título, parágrafos e links

### Encontro 2
- Tema: Estrutura HTML e semântica básica
- Prática: reorganizar página usando `header`, `main`, `section`, `footer`
- Produto: página semântica inicial

### Encontro 3
- Tema: Conteúdo rico (listas, tabelas e mídia)
- Prática: montar página de portfólio com lista de projetos e tabela de habilidades
- Produto: versão 1 do portfólio HTML

### Encontro 4
- Tema: Formulários
- Prática: criar formulário de contato com `label`, `input`, `textarea`, `button`
- Produto: formulário navegável por teclado

### Encontro 5
- Tema: Acessibilidade e SEO básico
- Prática: revisão por checklist (`alt`, hierarquia de títulos, links descritivos)
- Produto: página revisada e mais acessível

### Encontro 6
- Tema: Consolidar HTML
- Prática: mini-projeto "Página institucional estática"
- Produto: entrega da unidade

## Estratégias de Ensino
- Demonstração de código em tempo real.
- Correção coletiva com análise de diferentes soluções.
- Revisão por pares com checklist simples.

## Evidências de Aprendizagem
- Uso correto de elementos semânticos.
- Formulários consistentes e acessíveis.
- Estrutura clara de conteúdo para leitura humana e máquina.
