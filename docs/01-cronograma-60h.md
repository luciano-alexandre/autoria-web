# Cronograma da Disciplina (60h)

Referência: 20 encontros de 3h cada (ajustável para outro formato).

## Distribuição por Encontro

| Encontro | Carga | Tema Central | Entregável |
|---|---:|---|---|
| 1 | 3h | Introdução à Web, internet x web, cliente-servidor, ferramentas | Ambiente configurado |
| 2 | 3h | HTML: estrutura básica, tags essenciais | Página pessoal simples |
| 3 | 3h | HTML semântico: header, nav, main, section, article, footer | Estrutura semântica |
| 4 | 3h | Links, listas, tabelas, mídia e formulários | Formulário funcional |
| 5 | 3h | Acessibilidade inicial (alt, labels, landmarks) + SEO básico | Revisão acessível de página |
| 6 | 3h | CSS: sintaxe, seletores, cascata, box model | Página estilizada |
| 7 | 3h | Tipografia, cores, unidades, variáveis CSS | Guia visual de estilos |
| 8 | 3h | Layout com Flexbox | Landing page em Flexbox |
| 9 | 3h | Layout com Grid | Página com grid responsivo |
| 10 | 3h | Responsividade: media queries e mobile-first | Página adaptável |
| 11 | 3h | Componentização de interface + estados de interação | Biblioteca de componentes |
| 12 | 3h | Framework CSS (Bootstrap/Tailwind) - visão introdutória crítica | Releitura de layout com framework |
| 13 | 3h | JavaScript: sintaxe base, variáveis, tipos, operadores | Exercícios de lógica JS |
| 14 | 3h | Condicionais, laços e funções | Lista de funções |
| 15 | 3h | DOM: seleção, alteração de conteúdo e estilo | Página dinâmica simples |
| 16 | 3h | Eventos, formulários e validação no front-end | Formulário validado |
| 17 | 3h | DOM avançado: criação/remoção de nós, classes e atributos | To-do list funcional |
| 18 | 3h | Projeto integrador - desenvolvimento assistido | Versão beta do projeto |
| 19 | 3h | Projeto integrador - refinamento, testes e acessibilidade | Versão candidata |
| 20 | 3h | Apresentações, retrospectiva e fechamento | Projeto final + pitch |

## Marcos de Avaliação
- Semana 2 (Encontro 6): Checklist HTML + Acessibilidade.
- Semana 4 (Encontro 12): Mini-projeto CSS responsivo.
- Semana 6 (Encontro 17): Mini-projeto JS/DOM.
- Semana 7 (Encontro 20): Projeto integrador final.

## Plano de Contingência
- Se houver suspensão de aula: converter encontro em atividade assíncrona com roteiro + exercício de reposição.
- Se laboratório indisponível: atividades unplugged (planejamento de estrutura, wireframes, revisão de código impresso, pseudocódigo JS).
