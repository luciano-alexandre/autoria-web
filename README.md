# Disciplina: Autoria Web (60h)

Material base em formato Markdown para a disciplina de **Autoria Web** do Curso Técnico Integrado em Informática.

## Objetivo do Repositório
Organizar um roteiro completo de ensino com:
- ementa e objetivos de aprendizagem;
- cronograma de 60h;
- planos de aula por unidade;
- exercícios práticos e projetos;
- critérios de avaliação;
- referências (sites, documentação e videoaulas).

## Sugestão de Uso
1. Iniciar por `00-visao-geral.md` para alinhar objetivos.
2. Usar `01-cronograma-60h.md` para planejamento semanal.
3. Conduzir cada encontro com base em `02-roteiro-sequencial.md` + plano da unidade.
4. Aplicar atividades e projetos progressivamente.
5. Registrar evidências e notas usando `01-avaliacao-rubricas.md`.
